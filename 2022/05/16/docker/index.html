2.docker使用

（1）目前用的方法：

​ 搜索镜像：docker search # 在docker index中搜索image

​ 拉取基础镜像(从镜像库拉取）：docker pull [OPTIONS] NAME[:TAG]

​ docker pull tensorflow/tensorflow:1.15.4-gpu-py3

​ 定位到需要创建镜像的文件，根据dockerfile创建镜像：docker build [OPTIONS] PATH | URL

​ docker build -t resnet50:v1 .

​ 创建容器并启动：docker run [OPTIONS] IMAGE [COMMAND] [ARG…]

​ docker run -it -p 8080:8080 resnet50:v1

显示本地镜像：docker images

显示本地容器：docker ps [OPTIONS]

打包镜像：docker save [OPTIONS] IMAGE [IMAGE…]

​ docker save skfhfkshff -o /home/zhang/test.tar

删除镜像： docker rmi 镜像ID

删除容器: docker rm 容器ID
